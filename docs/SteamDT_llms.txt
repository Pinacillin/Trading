## API Docs
- CS2饰品信息 > 饰品价格相关接口 [通过marketHashName查询饰品价格](https://doc.steamdt.com/278832830e0.md): 获取全平台饰品价格、求购等数据。
- CS2饰品信息 > 饰品价格相关接口 [通过marketHashName批量查询饰品价格](https://doc.steamdt.com/278832831e0.md): 批量查询饰品价格、求购等数据
- CS2饰品信息 > 饰品价格相关接口 [查询steam饰品k线数据](https://doc.steamdt.com/428124801e0.md): 查询饰品K线数据(不包含成交数据)
- CS2饰品信息 > 饰品价格相关接口 [通过MarketHashName查询所有平台近7天均价](https://doc.steamdt.com/319748133e0.md): marketHashName 为Steam官方饰品名称，可通过获取Steam饰品基础信息接口获得
- CS2饰品信息 > 饰品磨损及检视图 [通过检视链接查询磨损度相关数据](https://doc.steamdt.com/273806087e0.md): 
- CS2饰品信息 > 饰品磨损及检视图 [通过ASMD参数查询磨损度相关数据](https://doc.steamdt.com/273806088e0.md): 
- CS2饰品信息 > 饰品磨损及检视图 [通过检视链接生成检视图,前提是已经获取到饰品的磨损度](https://doc.steamdt.com/273806089e0.md): 
- CS2饰品信息 > 饰品磨损及检视图 [通过ASMD参数生成检视图,前提是已经获取到饰品的磨损度](https://doc.steamdt.com/273806090e0.md): 
- CS2饰品信息 > CS2大盘数据 [查询大盘k线数据](https://doc.steamdt.com/450452402e0.md): 
- CS2饰品信息 > CS2大盘数据 [查询大盘最新指数数据](https://doc.steamdt.com/450452403e0.md):
