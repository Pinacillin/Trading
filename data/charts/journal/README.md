# Chart Analysis Journal

Use this folder for TradingView / Thinkorswim / MT4/MT5 chart analysis records.

- `chart_analyses.csv`: structured index for all chart analyses.
- Markdown records are created by `scripts/new_chart_analysis.py`.
- Add screenshots under `data/charts/screenshots/`.
- Add OHLCV CSV files under `data/charts/ohlcv/`.

Do not store exchange passwords, API secrets, browser cookies, or account session data here.
